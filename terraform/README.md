Terraform is a platform-agnostic IaC tool.

## commands
This command initializes the working directory containing Terraform configuration files.
```bash
terraform init
```
This command creates an execution plan. It shows what Terraform will do when you call apply. It does not deploy anything.
```bash
terraform plan
```
This command executes the actions proposed in a Terraform plan.
```bash
terraform apply
```
This command destroys the Terraform-managed infrastructure.
```bash
terraform destroy
```
